🚀 PRÓXIMOS PASOS (En orden)
═══════════════════════════════════════════════════════════════════════════════

PASO 1: PRUEBA LOCAL (10 minutos)
─────────────────────────────────────────────────────────────────────────────
1. Instala Node.js si no lo tienes: https://nodejs.org/
2. Abre terminal en esta carpeta
3. Ejecuta: npm install
4. Copia .env.example a .env
5. Edita .env con tu email de Gmail
6. Ejecuta: npm start
7. Abre: http://localhost:3000
8. Prueba el formulario

PASO 2: OBTÉN CONTRASEÑA DE APP GMAIL (2 minutos)
─────────────────────────────────────────────────────────────────────────────
1. Ve a: https://myaccount.google.com/apppasswords
2. Selecciona:
   - App: Mail
   - Device: Windows PC (o lo que uses)
3. Copia la contraseña de 16 caracteres
4. Pégala en EMAIL_PASSWORD de tu .env

PASO 3: PUBLICA EN INTERNET (5 minutos)
─────────────────────────────────────────────────────────────────────────────
Lee el archivo DEPLOY_RENDER.md y sigue estos pasos:
1. Crea repositorio en GitHub
2. Sube los archivos
3. Crea cuenta en Render.com
4. Conecta tu repositorio
5. Agrega variables de entorno
6. ¡Deploy! ✅

Resultado: Tu sitio estará en https://tu-app.onrender.com
